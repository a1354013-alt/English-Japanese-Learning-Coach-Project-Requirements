## v1.3.0 - Learning Intelligence Release

### Highlights

- Textbook-style English/Japanese lesson units.
- Item-level SRS for vocabulary, grammar, and sentence patterns.
- Snowball learning context using weak and recent items.
- Feynman feedback with AI provider support and deterministic fallback.
- Vocabulary roots, categories, tags, memory tips, and Excel import support.
- Deterministic demo reset for portfolio presentation.

### Known limitations

- TTS is provider-ready but disabled by default.
- Immersion is text shadowing only.
- Real recording and speech comparison are not included.
- RAG is optional and requires extra dependencies.
- This is a local single-user demo, not production multi-user SaaS.
